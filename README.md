# 楽天ROOMトラッカー（公開版）

楽天ROOMのフォロワー数・投稿数・いいね数をiOSショートカットで毎日記録し、GASのWebアプリでグラフ表示するダッシュボードです。

## 構成

```
Code.js           — GAS バックエンド（データ受取・スプレッドシート書込・ダッシュボード配信）
Index.html        — ダッシュボード UI
ios_shortcut_js.js — iOSショートカット内で実行するJSスクリプト
appsscript.json   — GAS マニフェスト
```

## セットアップ手順

### 1. スプレッドシートを作成する

Google スプレッドシートを新規作成する。シート名はどれでもよい（GASが自動作成します）。

### 2. GASプロジェクトを作成する

1. スプレッドシートのメニューから「拡張機能」→「Apps Script」を開く
2. `Code.gs` の中身を `Code.js` の内容に置き換える
3. ファイルメニューから「新しいファイル」→「HTML」で `Index.html` を作成し、内容を貼り付ける
4. ファイルメニューから「新しいファイル」→「JSON」で `appsscript.json` を作成（または既存のものを置き換え）し、内容を貼り付ける

### 3. WebAppとしてデプロイする

1. 右上の「デプロイ」→「新しいデプロイ」
2. 種類：「ウェブアプリ」
3. 次のユーザーとして実行：「自分（あなたのアカウント）」
4. アクセスできるユーザー：「全員（匿名ユーザーを含む）」
5. 「デプロイ」ボタンを押す
6. 表示されたWebアプリURLをメモする（iOSショートカットで使用します）

### 4. APIキーを設定する（任意・推奨）

スプレッドシートの「設定」シートに以下を追加すると、自分以外からのデータ送信を防げます。

| キー | 値 |
|------|----|
| api_secret_key | （任意の文字列。例: mysecret123） |

### 5. iOSショートカットを作成する

以下のアクションをこの順番で組み合わせます。

#### アクション構成

1. **Safariでページを開く**
   - URL: あなたの楽天ROOMプロフィールURL（例: `https://room.rakuten.co.jp/room_xxx`）

2. **少し待つ**
   - 1〜2秒待機（ページ読み込み完了のため）

3. **Webページ上でJavaScriptを実行**
   - `ios_shortcut_js.js` の内容をそのまま貼り付ける

4. **テキストを辞書に変換**
   - 前のアクションの結果を辞書（JSON）に変換

5. **URLの内容を取得**
   - URL: `（手順3でメモしたWebアプリURL）`
   - 方式: POST
   - ヘッダー: `Content-Type: application/json`
   - 本文（JSON）:
     ```json
     {
       "action": "receiveStats",
       "api_key": "（設定したapi_secret_key）",
       "followers": 「辞書」の「followers」,
       "following": 「辞書」の「following」,
       "posts": 「辞書」の「posts」,
       "likes": 「辞書」の「likes」,
       "url": 「辞書」の「url」
     }
     ```
   - ※ api_secret_key を設定しない場合は `"api_key"` 行は不要

6. **（オプション）通知を表示する**
   - URLの内容を受け取り結果として表示

#### 自動実行の設定

ショートカットの「オートメーション」から「毎日 ○○時に実行」を設定することで、自動でデータが記録されます。

### 6. ダッシュボードを確認する

WebアプリのURLをブラウザで開くとダッシュボードが表示されます。ホーム画面に追加しておくと便利です。

## データスキーマ

スプレッドシートの「記録」シートに以下の形式で記録されます（1日1行、同日は上書き）。

| 日時 | URL | フォロワー数 | フォロー数 | 投稿数 | いいね数 | ランク |
|------|-----|------------|----------|-------|--------|------|

## トラブルシューティング

**データが記録されない**
- ショートカットのPOST先URLが正しいか確認
- `api_secret_key` を設定した場合、ショートカット側の `api_key` と一致しているか確認
- GASのデプロイ設定で「全員（匿名ユーザーを含む）」になっているか確認

**グラフに何も表示されない**
- ダッシュボードの「↻ 更新」ボタンを押す
- iOSショートカットを一度手動実行してデータを送信する

**フォロワー数が取得できない**
- `ios_shortcut_js.js` の DOM抽出パターンが楽天ROOMのHTML構造と合っていない可能性があります
- ショートカットでJavaScriptを実行した後の戻り値を確認し、`null` になっているキーがあれば正規表現を調整してください
